## ğŸ§  Machine Learning Approach

  - Data augmentation
  - Batch normalization

### Key Metrics (Validation/Test Set)

- **Accuracy:** ~87.6%
- **F1-score (malignant):** ~0.88
- **Recall / Sensitivity (malignant):** ~88.4%
- **ROC-AUC:** ~0.95
- **PR-AUC:** ~0.95

These results indicate **strong discriminative capability on the evaluation dataset**, while acknowledging real-world limitations.
